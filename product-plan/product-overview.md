# SnapStim

## Description

SnapStim is an ABA skill-acquisition tool that reduces the time and friction BCBAs spend building, organizing, and running programs, while maintaining clinical quality and data integrity. It improves team consistency across staff and devices and produces cleaner session data for clinical decision-making.

## Problems & Solutions

### Problem 1: Time-Consuming Program Creation

AI-assisted program creation with stimulus generation across 4 program types eliminates manual work.

### Problem 2: Inconsistent Implementation

Standardized trial flow with consistent data capture regardless of who runs the session.

### Problem 3: Messy Session Data

Clean trial-by-trial outcome data with automatic recap generation for clinical decisions.

### Problem 4: Tedious Content Review

Human-in-the-loop review screen with regenerate, enlarge, and edit capabilities.

### Problem 5: Communication Gaps

Role-separated program-level comments with appropriate privacy boundaries.

## Key Features

- Role-based accounts and permissions (BCBA, RBT, Caregiver)
- Client profiles and program management with auto-assign on save
- Program creation with AI-powered stimulus generation
- Human-in-the-loop review and regeneration
- Session running with consistent trial flow and data capture
- Communication and reporting with role-based privacy
